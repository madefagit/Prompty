## Wyglad i UX — Kompletny redesign

Improve the entire visual design system of my Angular application and refactor the UI to look like a modern premium SaaS platform used in logistics / enterprise operations.

Current app feels outdated, flat and inconsistent. I want a complete visual upgrade across the whole application while keeping all existing Angular logic, routes, services and functionality unchanged.

MAIN GOAL:
Transform the app into a polished, modern, premium admin system (2026 standard) similar to top SaaS dashboards.

==================================================
GLOBAL DESIGN SYSTEM
==================================================

Create a consistent design system for the whole app:

1. Colors
- Modern dark theme with elegant contrast
- Neutral dark backgrounds
- Soft borders
- Primary accent color: green or emerald
- Success / warning / error semantic colors
- Accessible contrast ratios

2. Typography
- Clean professional font styling
- Strong page titles
- Clear section headers
- Smaller muted helper text
- Better line-height and spacing

3. Spacing
- Use consistent spacing scale (8 / 12 / 16 / 24 / 32)
- Remove cramped layouts
- Add breathing room everywhere

4. Borders / Radius / Shadows
- rounded-xl cards
- soft shadows
- subtle borders
- premium modern depth

==================================================
LAYOUT IMPROVEMENTS
==================================================

1. Sidebar
- modern collapsible sidebar
- active menu highlight
- icons for navigation
- cleaner spacing
- hover states

2. Topbar
- page title
- breadcrumbs
- search input
- notifications area
- user profile dropdown

3. Main Content
- max width containers
- proper padding
- clean section separation

==================================================
TABLES / DATA GRIDS
==================================================

Upgrade all tables:

- sticky headers
- zebra hover rows
- row hover effect
- better spacing
- cleaner pagination
- filters styled properly
- search above table
- row actions with icon buttons
- responsive behavior

==================================================
FORMS
==================================================

Upgrade all forms and modals:

1. Inputs
- larger height
- rounded corners
- borders
- focus ring
- placeholders
- labels above fields

2. Selects / dropdowns
- modern styling
- searchable if possible

3. Validation
- inline errors
- red helper text
- proper required indicators

4. Forms layout
- 2-column responsive grid desktop
- 1-column mobile

==================================================
MODALS / DRAWERS
==================================================

Replace outdated modals with premium dialogs:

- rounded corners
- proper width
- better spacing
- clear header
- footer action buttons
- support right-side drawer where useful

==================================================
BUTTONS
==================================================

Standardize buttons:

- Primary = green
- Secondary = dark outlined
- Danger = red
- Ghost = transparent

Use consistent sizes:
- sm / md / lg

==================================================
DASHBOARD
==================================================

Improve dashboard pages:

- KPI cards
- charts area
- recent activity
- quick actions
- status widgets

==================================================
MICRO INTERACTIONS
==================================================

Add subtle polish:

- hover transitions
- smooth open/close animations
- loading skeletons
- button states
- empty states

==================================================
RESPONSIVE
==================================================

Ensure full mobile/tablet support.

==================================================
TECHNICAL RULES
==================================================

- Keep Angular architecture unchanged
- Keep TypeScript business logic unchanged
- Only improve UI / UX / templates / styles
- Reuse components where possible
- Use Tailwind CSS if available
- If PrimeNG / Angular Material exists, restyle professionally
- Refactor repeated styles into reusable shared components

==================================================
DELIVERABLES
==================================================

1. Apply redesign across entire app
2. Create reusable design tokens/components
3. Clean inconsistent legacy styles
4. Improve visual hierarchy everywhere
5. Make app feel premium and production-ready

DESIGN REFERENCE
==================================================

Think:
Linear + Notion + modern ERP + enterprise logistics SaaS + 2026 UI quality

When user opens app, first impression should be:
"This looks expensive, modern and trustworthy."
